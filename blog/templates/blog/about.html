About Page

  • This is a simple blog portal
  • This project is done using django framework
  • This portal has options like
    • Register
    • Login
    • Has a home page and about informations
    • Home page contains all post with
      1. post title
      2. post content
      3. post creation information
      4. The post listed in the order of date posted

    • Once logged in they can:
      • See all their own post ordered in posting date
      • Create new post
      • Update and delete an existing post
      • Update the profile information
      • This profile information portal has been left undeveloped for further developement
